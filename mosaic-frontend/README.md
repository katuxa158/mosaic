# Mosaic

Frontend-часть проекта **Mosaic** — это одностраничное приложение (SPA),
реализующее пользовательский интерфейс

## ⚙️ Установка и запуск
### 1. Клонирование репозитория
```bash
git clone <URL_репозитория>
cd mosaic-frontend
```

### 2. Установка зависимостей
```bash
npm install
```

### 3. Запуск приложения
```bash
npm run dev
```
